# note

> 20年资深程序员的技术知识库

一个涵盖编程语言、AI/ML、网络协议、数据库、开发工具等领域的技术文档库。

## 📚 内容结构

| 分类  | 说明 |
|------|------|
| **编程语言**  | Go标准库、Python标准库、框架实战 |
| **AI/机器学习** |  | 大模型、算法、AI Agent |
| **网络协议** |  | HTTP/2/3、gRPC、WebRTC、IPv8 |
| **开发工具** |  | Docker、Kubernetes、Linux、Git |
| **数据库** |  | MySQL优化、B+树 |
| **计算机科学** |  | 计算机组成、数学基础 |

## 🚀 快速开始

### 本地预览

```bash
# 安装依赖
pip install mkdocs mkdocs-material

# 进入项目目录
cd note

# 启动开发服务器
mkdocs serve
```

### 构建静态站点

```bash
mkdocs build
```

## ✨ 技术栈

- **MkDocs** - 文档生成器
- **Material Theme** - 现代化主题
- **Mermaid** - 图表支持
- **Pygments** - 代码高亮

## 📖 使用指南

1. 克隆仓库到本地
2. 安装MkDocs及依赖
3. 使用 `mkdocs serve` 预览文档
4. 浏览器访问 `http://localhost:8000`
